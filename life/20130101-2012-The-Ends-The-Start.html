2012 The End, The Start

2013-01-01 日记

《2012》圆满度过了,我们都是幸存者,世界没有走到尽头,我们没有看到末日.

2012全年都是紧张的度过的吧.起初,每天到学校看到倒计时牌写着距离高考的剩余时间；然后,每天起床数着距离进入大学的时间,想要充实的过完＂最后一个＂暑假；最后等待世界末日的到来.
好在玛雅人和我们开了玩笑,让我有了下一个暑假,让我可以继续人生,让这个恐惧的心情有了一个Happy Ending.

回眸2012,半个高三的生活还是充满激情的,每天固定的吃饭时间,固定的晚餐,固定的作息,机械的作息时间表当然不会记载趣事：

还记得晚自习课间偷偷催同学拿出iPhone4S,然后围着它调戏Siri.

还记得做题做烦的时候谈谈暑假计划.

还记得午休和考试前的休息时间,我就像个推销员似的挨个推销Apple产品.

还记得和两位密友去吃吃小吃暴暴粗口,顺便讨论学习大计.

还记得互相大叫＂用功帝＂,＂哇,都做完了＂,＂你就疯去吧＂,＂人类已经无法阻止你了＂……这一类为自己加油,顺便夸奖他人的话.

半个高三短暂,但是过的异常充实,即使记得也写不下了.

至于暑假,就颓着呗,当作最后一个假期,怎能不好好休闲一番.写了个飞行棋游戏,还不能战斗；旅游能吸引我的不是风景,而是打扑克；在家里做了食物给同学们吃,结果很难吃……反正是颓废颓废嘛,结果如何不重要.

一晃,暑假没了,进入＂天堂＂——大学.两周军训感觉好辛苦,好痛苦,鞋子特别薄,我的脚都要爆掉了,吃饭还要一个排一个排轮流.教官还特别喜欢刁难我们,所以训练的时候总想把教官做掉,要他再敢欺负我们.训练的过于劳累,导致不想吃饭,于是每天的伙食费低得惊人,感觉满自豪的.

两周的烂军训结束后,真正开始体会＂天堂＂生活.过着过着就想大骂：呸！这算哪门子天堂,还是很苦嘛！不过相比较高中初中,真的轻松多啦,没有满课,没有每天都交作业,周末也是完整的,真的算是高中的天堂了.

然后进入篮球赛,小组赛全败,也对,我们都不够高,身高拼不过,体重也拼不过,那我们班有什么优势呢？哦对,我们女生最多……女生多有什么效果呢？很显然：男生少了,这样要出篮球高手的概率也就低了.

在这偏僻的小村庄学习,吃不到意大利面,麦当劳,吃不起肉,于是就只能拮据一周,周末去大撮一顿,恨不得把下一周的所有空间吃了.

每天晚上都和爸妈发几条信息,不是思念只是随便聊聊,因为进入大学便是准备好与父母分开的Start.

如果一学期不见他们,突然看见的话,衰老会很明显,但是如果时不时来个Facetime,就不会有一种瞬间苍老的感觉.说到这,就一种感激之情,关爱之情涌上来；真的他们老了,他们的辉煌要end了,我们的辉煌要start了,是时候多关心他们了,多为他们着想了.

打开QQ空间看着今年写下的日志,感觉自己真的成长了不少,以前回忆过去只会觉得原来的自己真的好白痴,竟然会做这么傻的事情.而今年,真的算是成熟了.这一年的日志有励志的,有纪录心情的,不像一年以前那样,发一堆小儿科的小把戏,小发现,小科技在空间里了.

2012年在成都留下了我的足迹,也把我的朋友圈从浙江扩大到全国,让我知道了什么叫做独立生活.我体验了赤字的生活,也体验了小资生活,有巨勤奋的经历,也有浑浑噩噩的日子.我不用像《北京青年》一样重走青春路,因为这半年我已经体验过大部分青春啦.

玛雅人的预言或许改变了这一年我们对生活的态度,让我们不敢去浪费每一天,把每一天都当最后一天去度过；或许也改变了那些已经离我们而去的人在临终前的心态；或许改变了整个世界,让旧的世界走向末日.

2013即将到来,虽然那个令人悚然的预言没有了,但是危险还是处处存在的,平平安安过完这一年这是每个人的愿望.而这平静的一年一定会因我们的存在变的不平静.2013 Now Start！

2012,这一页翻过去了,伴随着电影的恐慌,记载着学习生活的欢乐与紧张,承载这辛酸苦楚.它会变成一页黄纸留在心中,最后留下一句：The End.
